## 🔴 Un vrai problème à corriger dans la sidebar

La sidebar a les noms d'utilisateur **codés en dur** :

```python
ui.label('Johanna Doe').classes('pm-drawer-name')
ui.label('johanna@company.com').classes('pm-drawer-mail')
ui.avatar('https://i.pravatar.cc/120?img=32')
```

Ce n'est pas bloquant mais c'est visible. Voici la correction rapide pour `sidebar.py` :

```python
# app/components/sidebar.py — section profil à remplacer

from app.core.session import SessionManager   # ← ajouter cet import en haut

# Remplacer le bloc pm-drawer-profile par :
with ui.element('div').classes('pm-drawer-profile'):
    ui.icon('account_circle').style('font-size: 72px; color: #c3c9d1;')
    full_name = SessionManager.get_full_name() or 'Utilisateur'
    email     = SessionManager.get_email()    or ''
    ui.label(full_name).classes('pm-drawer-name')
    if email:
        ui.label(email).classes('pm-drawer-mail')
```
